# Template FULL STACK WEB APP USING CLOUDFLARE

## Setup

1. yarn install
2. wrangler login
3. cp workers/.env.example workers/.env
4. cp pages/.env.example pages/.env
5. edit .env and workers/wrangler.toml
6. cd pages && yarn wrangler pages project create ${YOUR_PAGES_PROJECT}

Use devcontainer as you like.
